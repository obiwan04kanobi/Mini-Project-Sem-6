About the Project

The CIFAR-100 Image Classifier is a web application built using a Convolutional Neural Network (CNN) trained on the CIFAR-100 dataset. It supports real-time image predictions for 100 distinct categories, including animals, vehicles, and household objects. The backend is powered by a Flask server, with a pre-trained model implemented using TensorFlow, while the frontend leverages Tailwind CSS for a clean and responsive design.

The model has been trained to achieve high accuracy, ensuring reliable predictions. This application showcases the integration of machine learning in web development, emphasizing user-friendly functionality for non-technical users. Whether you're exploring deep learning or need a quick image classifier, this project demonstrates the potential of AI-driven solutions.
